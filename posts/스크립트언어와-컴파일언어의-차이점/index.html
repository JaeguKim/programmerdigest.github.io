스크립트언어와 컴파일언어의 차이점

Jaegoo.Kim on Jul 12, 20172017-07-12T00:00:00+08:00
Updated Mar 102021-03-10T16:08:56+08:00 1 min read

스크립트 언어

  • php, javascript, python …

  • 컴파일 과정을 거치지 않고 실시간으로 텍스트를 분석하며 실행

  • 코드에 문법 오류 가있을 경우 실행시점에 발견됨

  • 스크립트 엔진이 포팅되어 있는 모든 운영체제에서 실행 가능

  • 컴파일 언어보다 실행속도가 느림

컴파일 언어:

  • 컴파일러에 의해 기계어로 번역된 채로 실행됨

  • 문법오류가 컴파일 시점에 발견됨

  • 목표로 한 운영체제에서만 실행가능

  • 실행이 빠르나, 수정이 빈번하게 발생할 경우, 수정 후 일일히 컴파일을 다시 해야됨.
